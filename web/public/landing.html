// How it works
你做一件事，系統做所有事
從目標到交付，全部自動化。卡住了？系統自己修復。
01

定義目標

你用自然語言描述你要什麼。一句話就夠。

👤 唯一需要你做的事
02

AI 自動拆解

GoalDecomposer 用 Claude 把目標拆成 milestone 和可執行任務。

🧠 Claude Sonnet
03

智能分配

TaskRouter 根據 Agent 能力匹配與負載平衡，自動分配給最適合的 Agent。

⚡ 每 5 分鐘
04

Agent 執行

Agent 帶入知識庫歷史經驗，Claude 驅動的心跳執行器持續推進任務。

🧠 Knowledge-Augmented
05

卡住？自愈

AutonomousLoop 偵測到任務阻塞，自動產生替代策略並重新分配。

⚡ 每 10 分鐘
06

完成歸檔 & 通知

目標完成後自動歸檔進公司檔案庫，Telegram 推播成果。你繼續喝咖啡。

⚡ 自動觸發

// CEO Hiring
CEO 雇用員工，直接指派任務
CEO Agent 不只執行任務，還能雇用新員工、指派工作、監控組織架構。真正的零人公司運作模式。
🤝
AI 生成招募提示
CEO 輸入職責描述，Claude 自動生成完整系統提示，Agent 立即上線工作。
POST /api/agents/hire
{
  "name": "Jane",
  "role": "marketing",
  "responsibilities": "..."
}
→ AI 生成系統提示
→ 歸入公司檔案庫
→ Telegram 通知
🎯
CEO 直接指派任務
繞過自動路由，CEO 直接指定哪個 Agent 做哪件事，系統自動檢查工作量上限。
POST /api/tasks/:id/assign
{
  "agentId": "founding-eng",
  "note": "優先處理"
}
→ 檢查 Agent 工作量
→ 記錄指派留言
🏢
組織架構圖
即時可視化公司組織，CEO 在頂端，直屬員工展開顯示，每個 Agent 的負載一目了然。
